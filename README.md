# Materi Perkuliahan

Repositori ini berisi kumpulan materi pembelajaran untuk beberapa mata kuliah di bidang teknologi informasi dan rekayasa komputer.

## Mata Kuliah
Materi yang tersedia dalam repositori ini meliputi:

1. Probabilitas dan Statistika
2. Teknik Penambangan Data
3. Teknologi Cloud
4. Arsitektur Komputer dan Sistem Operasi

## Isi Materi
Setiap mata kuliah terdiri dari beberapa komponen pembelajaran, antara lain:
- Slide perkuliahan
- Modul praktikum
- Tugas mahasiswa
- Dataset dan referensi pendukung

Repositori ini bertujuan untuk mendukung proses pembelajaran serta memudahkan mahasiswa dalam mengakses materi kuliah secara terstruktur.
