# Handout Arsitektur Komputer dan Sistem Operasi

Folder ini berisi kumpulan **handout materi perkuliahan** untuk mata kuliah **Arsitektur Komputer dan Sistem Operasi**. Handout disediakan sebagai bahan pendukung pembelajaran untuk membantu mahasiswa memahami konsep dasar arsitektur komputer dan sistem operasi.

## Cakupan Materi
Handout dalam folder ini dapat mencakup beberapa topik berikut:

1. Pengantar Arsitektur Komputer
2. Struktur dan Fungsi CPU
3. Sistem Memori
4. Perangkat Input dan Output
5. Pengantar Sistem Operasi
6. Manajemen Proses
7. Manajemen Memori
8. Sistem Berkas (File System)
9. Manajemen Perangkat I/O
10. Keamanan Sistem Operasi

## Tujuan
Handout ini bertujuan untuk:
- memberikan ringkasan konsep utama yang dibahas dalam perkuliahan
- membantu mahasiswa mempersiapkan diri sebelum mengikuti kuliah
- menjadi referensi tambahan dalam memahami materi perkuliahan

## Catatan
Mahasiswa disarankan untuk membaca handout ini sebelum mengikuti perkuliahan agar proses pembelajaran dapat berlangsung lebih efektif.
