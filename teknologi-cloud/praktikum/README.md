
# Praktikum Teknologi Cloud

Folder ini berisi panduan **praktikum dan latihan implementasi** untuk mata kuliah **Teknologi Cloud**.

## Kegiatan Praktikum
Praktikum dapat mencakup beberapa kegiatan berikut:

- Pengenalan platform cloud
- Pembuatan instance server cloud
- Konfigurasi layanan web pada cloud
- Manajemen penyimpanan cloud
- Deploy aplikasi pada cloud
- Monitoring dan manajemen layanan cloud

## Tools yang Digunakan
Praktikum dapat menggunakan beberapa platform seperti:

- AWS
- Google Cloud
- Microsoft Azure
- Docker dan container
