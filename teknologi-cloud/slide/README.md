# Slide Perkuliahan Teknologi Cloud

Folder ini berisi materi **slide perkuliahan** untuk mata kuliah **Teknologi Cloud**.

## Cakupan Materi
Beberapa topik yang dibahas dalam slide antara lain:

1. Pengantar Cloud Computing
2. Model Layanan Cloud (IaaS, PaaS, SaaS)
3. Model Deployment Cloud (Public, Private, Hybrid Cloud)
4. Virtualisasi dan Container
5. Arsitektur Cloud
6. Manajemen Infrastruktur Cloud
7. Keamanan pada Cloud Computing
8. Implementasi layanan cloud menggunakan platform cloud modern

## Tujuan
Slide digunakan sebagai media utama dalam penyampaian materi untuk membantu mahasiswa memahami konsep dan arsitektur teknologi cloud.
