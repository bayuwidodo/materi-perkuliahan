# Project Teknologi Cloud

Folder ini berisi **project atau tugas akhir** pada mata kuliah **Teknologi Cloud**.

## Tujuan Project
Project bertujuan untuk memberikan pengalaman kepada mahasiswa dalam merancang dan mengimplementasikan solusi berbasis cloud.

## Contoh Project
Beberapa contoh project yang dapat dilakukan antara lain:

- Deploy aplikasi web pada cloud
- Implementasi arsitektur cloud sederhana
- Integrasi database dan server pada cloud
- Implementasi container menggunakan Docker
- Monitoring dan manajemen layanan cloud

Mahasiswa diharapkan mampu mengintegrasikan konsep yang telah dipelajari dalam bentuk implementasi nyata menggunakan platform cloud.
