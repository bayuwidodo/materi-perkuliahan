# Handout Praktikum Teknologi Cloud

Folder ini berisi **handout atau panduan praktikum** untuk mata kuliah **Teknologi Cloud**.

Handout berfungsi sebagai panduan langkah-langkah praktikum yang membantu mahasiswa dalam melakukan implementasi teknologi cloud.

## Materi Handout
Handout praktikum dapat mencakup:

- Panduan membuat virtual machine pada cloud
- Konfigurasi server pada cloud
- Deploy aplikasi web pada cloud
- Penggunaan container dan docker
- Manajemen resource cloud

## Tujuan
Handout ini bertujuan untuk membantu mahasiswa mengikuti kegiatan praktikum secara sistematis dan memahami implementasi teknologi cloud secara langsung.
