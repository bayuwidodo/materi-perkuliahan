# Praktikum Teknik Penambangan Data

Folder ini berisi panduan **praktikum dan latihan implementasi** untuk mata kuliah **Teknik Penambangan Data**.

## Kegiatan Praktikum
Praktikum dapat mencakup beberapa kegiatan berikut:

- Eksplorasi data
- Pembersihan dan preprocessing data
- Implementasi algoritma klasifikasi
- Implementasi algoritma clustering
- Analisis asosiasi (association rule)
- Evaluasi model data mining

Praktikum dapat menggunakan tools seperti:
- R
- Python
- RapidMiner
- WEKA
