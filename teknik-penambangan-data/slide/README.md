# Slide Perkuliahan Teknik Penambangan Data

Folder ini berisi materi **slide perkuliahan** untuk mata kuliah **Teknik Penambangan Data**.

## Cakupan Materi
Beberapa topik yang dibahas dalam slide antara lain:

1. Pengantar Data Mining
2. Tahapan Proses Data Mining
3. Preprocessing Data
4. Teknik Klasifikasi
5. Teknik Clustering
6. Association Rule Mining
7. Evaluasi Model
8. Implementasi Data Mining menggunakan R atau Python

## Tujuan
Slide digunakan sebagai media utama dalam penyampaian materi perkuliahan untuk membantu mahasiswa memahami konsep dan metode dalam penambangan data.
