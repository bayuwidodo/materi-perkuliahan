# Dataset Teknik Penambangan Data

Folder ini berisi **dataset yang digunakan dalam praktikum dan latihan** pada mata kuliah **Teknik Penambangan Data**.

## Jenis Dataset
Dataset yang tersedia dapat digunakan untuk berbagai teknik analisis, seperti:

- klasifikasi
- clustering
- asosiasi
- prediksi

## Tujuan
Dataset disediakan untuk membantu mahasiswa dalam:
- melakukan eksperimen algoritma data mining
- memahami proses analisis data
- menguji performa model data mining
