# Handout Teknik Penambangan Data

Folder ini berisi kumpulan **handout materi perkuliahan** untuk mata kuliah **Teknik Penambangan Data**. Handout disediakan sebagai bahan pendukung pembelajaran yang membantu mahasiswa memahami konsep, metode, dan penerapan teknik penambangan data.

## Cakupan Materi
Handout dalam folder ini dapat mencakup beberapa topik berikut:

1. Pengantar Data Mining
2. Tahapan Proses Data Mining (CRISP-DM)
3. Data Preprocessing
4. Teknik Klasifikasi
5. Teknik Clustering
6. Association Rule Mining
7. Evaluasi Model
8. Visualisasi dan Interpretasi Data
9. Implementasi Data Mining menggunakan R atau Python

## Tujuan
Handout ini bertujuan untuk:
- memberikan ringkasan konsep utama yang dibahas dalam perkuliahan
- membantu mahasiswa memahami metode analisis data
- menjadi referensi tambahan dalam kegiatan praktikum dan tugas

## Catatan
Mahasiswa disarankan untuk membaca handout sebelum mengikuti perkuliahan atau praktikum agar dapat memahami konsep yang akan dipelajari dengan lebih baik.
