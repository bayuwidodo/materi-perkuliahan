# Handout Probabilitas dan Statistika

Folder ini berisi kumpulan **handout materi perkuliahan** untuk mata kuliah **Probabilitas dan Statistika**. Handout disusun untuk membantu mahasiswa memahami konsep dasar probabilitas dan statistika yang digunakan dalam analisis data.

## Isi Handout
Beberapa topik yang dibahas dalam handout antara lain:

1. Pengantar Probabilitas dan Statistika
2. Konsep Ruang Sampel dan Kejadian
3. Aturan Dasar Probabilitas
4. Distribusi Peluang
5. Variabel Acak
6. Distribusi Normal
7. Statistika Deskriptif
8. Pengantar Inferensi Statistik

## Tujuan
Handout ini disediakan sebagai bahan pendukung pembelajaran yang dapat digunakan mahasiswa untuk:
- memahami konsep dasar probabilitas dan statistika
- mempersiapkan diri sebelum perkuliahan
- membantu dalam pengerjaan latihan dan tugas

## Catatan
Mahasiswa disarankan untuk mempelajari handout ini sebelum mengikuti perkuliahan agar proses pembelajaran dapat berjalan lebih efektif.
